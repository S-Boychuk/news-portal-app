# 📰 News Portal — Team JavaScript Application

A dynamic and modern web application that aggregates and filters world news using a REST API. Developed collaboratively to practice advanced vanilla JavaScript architecture and asynchronous data fetching.

## 👩‍💻 My Contribution (Core Logic & UI Components)
* **Calendar Integration:** Implemented an interactive date-picker input element and integrated a calendar librar for date selection.
* **Date-Based Filtering:** Engineered the JavaScript logic to filter incoming API news arrays dynamically based on the user's selected calendar date.
* **Search Fallback Interface:** Developed the "No Results Found" page.
* **Footer Component:** Built the responsive site Footer.

* **Date Boundary Testing:** Verified calendar behavior across edge cases;
* **Empty State Validation:** Tested the exact triggering conditions and rendering of the "No Results Found" interface across all viewports when the API returns 0 items.
* **Responsive Layout Stability:** Handled cross-browser and adaptive testing for the input field, calendar popover, and footer layout.
